Setup
Running in 60 seconds
01
Clone & install
One pip install — no API keys, no accounts, no configuration files.
git clone github.com/janmejai2002/unstop-mcp cd unstop-mcp pip install -r requirements.txt
02
Add to Claude Desktop
Paste one JSON block into claude_desktop_config.json and restart.
{ "mcpServers": { "unstop": { "command": "python", "args": ["server.py"] } } }
03
Start asking
